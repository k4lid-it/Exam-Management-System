# University Exam Management System

## Overview
A web-based system designed to computerize the university exam process, enabling both virtual and in-campus exam monitoring.

## Problem Statement
Traditional exam management relies heavily on manual processes, making monitoring, coordination, and reporting inefficient.

## Solution
The system provides a centralized platform for administrators and IT support staff to manage exams, monitor sessions, and handle issues in real time.

## Technologies Used
- React
- JavaScript
- SQL Database 
- WebSocket
- REST API
- QR codes 

## Key Features
- Fully computerised on-site exam monitoring
- Admin control dashboard
- Paperless invigilation cycle
- User management system
- Real-time issue tracking

## Challenges
Ensuring system reliability during peak exam times and handling simultaneous users.

## Results
The system improved exam coordination and reduced manual workload.

## Future Improvements
- AI-based cheating detection  
- Advanced analytics dashboard
